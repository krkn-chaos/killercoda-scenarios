# killercoda-scenarios

Interactive [Killercoda](https://killercoda.com) scenarios for the [Krkn](https://github.com/krkn-chaos/krkn) chaos engineering ecosystem.

## Scenarios

| Scenario | Description |
|----------|-------------|
| [krkn-quickstart](krkn-quickstart/) | Getting started with Krkn: run pod, node, and multi-scenario chaos with `krknctl` on a 2-node cluster, read exit codes, and watch chaos impact live in the krkn-visualize web UI (~25 min) |
| [krkn-operator-demo](krkn-operator-demo/) | Install OCM and krkn-operator for multi-cluster chaos engineering, expose the operator, and test its API |
