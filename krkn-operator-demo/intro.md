# Krkn Operator Demo

Welcome to this interactive demo of the **krkn-operator**!

In this scenario you will learn how to:

- Install Open Cluster Management (OCM) on a Kubernetes cluster
- Configure the krkn-operator for chaos engineering
- Expose the operator to access it from outside the cluster
- Verify the installation and test connectivity

The **krkn-operator** is a chaos engineering tool that allows you to run chaos experiments on multi-cloud Kubernetes clusters managed via OCM.

Let's get started!
